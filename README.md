Student Lifestyle, Mental Health and Burnout Dataset Analysis is a multivariate statistical analysis project focused on exploring patterns in student well-being, academic pressure, lifestyle behaviour, and mental health risk levels. Using a large Kaggle dataset containing 1,000,000 observations and 20 features, the study applies Principal Component Analysis (PCA) for dimensionality reduction, Factor Analysis (FA) to identify latent structures, and Quadratic Discriminant Analysis (QDA) to classify students into risk groups. The analysis highlights key dimensions such as psychological distress, academic pressure, digital behaviour, burnout and recovery, and social-emotional adjustment, while the QDA model demonstrates strong performance in classifying student risk levels.
